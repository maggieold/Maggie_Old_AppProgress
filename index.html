Greeting Generator

Description:

The follow is a generative web application that allows the user to create a grid of squares that is controlled by a graphical user interface. Use the panel on the right to get started.

Parameters:

  • Color: Change the colors of all things on screen
  • Squares: Change number of concentric squares
  • Distance: change spacing of concentric squares
  • Stroke Weight: Weight of square stroke
  • Roughness: Change the haphazardness of the squares
  • Rotate:rotate the grid along a central point
  • Translate X: Move the grid left and right
  • Translate Y: Move the grid up and down
  • Text: Input your own text
  • Text Size:change the text size
